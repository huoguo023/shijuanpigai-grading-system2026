【常见问题】━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Q1: 双击 BAT 文件没反应？
  A1: 右键 → 属性 → 解除锁定 → 确定

  Q2: 浏览器打不开网页？
  A2: 检查 HTML 文件是否存在，或直接双击 HTML 文件

  Q3: SVG 动画不显示？
  A3: 使用 Chrome 或 Edge 浏览器

  Q4: 音效没有声音？
  A4: 检查电脑音量，点击页面任意位置激活音频

  Q5: 可以离线使用吗？
  A5: 可以！所有功能都支持离线使用

  Q6: 如何分享给其他人？
  A6: 将整个文件夹压缩成 ZIP，分享下载链接

  Q7: 可以在手机上使用吗？
  A7: 可以！将文件传到手机，用浏览器打开 HTML

  Q8: 杀毒软件报警？
  A8: BAT 文件是安全的，可以添加到白名单

  Q9: 如何更新内容？
  A9: 下载新版本，替换旧文件即可

  Q10: 可以修改内容吗？
  A10: 可以！用文本编辑器打开 HTML 文件修改
