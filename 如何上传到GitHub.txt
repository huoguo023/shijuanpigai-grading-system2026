【上传步骤】━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  方法一：使用自动脚本（⭐ 推荐）
  
    1. 在 GitHub 创建新仓库
       • 访问: https://github.com/new
       • 填写仓库名称（如: elementary-science-grade4）
       • 选择 Public 或 Private
       • 不要勾选 "Initialize with README"
       • 点击 "Create repository"
    
    2. 双击运行脚本
       └─ 双击: 上传到GitHub.bat
    
    3. 输入仓库地址
       • 复制 GitHub 显示的仓库地址
       • 粘贴到脚本中
       • 示例: https://github.com/YOUR_USERNAME/YOUR_REPO.git
    
    4. 确认上传
       • 脚本会自动推送所有文件
       • 等待上传完成
    
    5. 完成！
       • 访问您的 GitHub 仓库
       • 查看所有文件

  ────────────────────────────────────────────

  方法二：使用命令行
  
    1. 在 GitHub 创建新仓库（同上）
    
    2. 打开命令提示符（CMD）
    
    3. 运行以下命令:
       
       # 添加远程仓库（替换为您的仓库地址）
       git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
       
       # 推送到 GitHub
       git branch -M main
       git push -u origin main
    
    4. 输入 GitHub 用户名和密码（或 Token）
    
    5. 完成！

  ────────────────────────────────────────────

  方法三：使用 GitHub Desktop
  
    1. 下载并安装 GitHub Desktop
       • 下载: https://desktop.github.com/
    
    2. 登录 GitHub 账号
    
    3. 添加本地仓库
       • File → Add Local Repository
       • 选择当前文件夹
    
    4. 发布到 GitHub
       • 点击 "Publish repository"
       • 填写仓库名称
       • 点击 "Publish"
    
    5. 完成！

【重要提示】━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ⚠️  首次推送可能需要登录
  
  如果提示需要认证:
  
  方法1: 使用 Personal Access Token
    1. GitHub → Settings → Developer settings
    2. Personal access tokens → Generate new token
    3. 选择 repo 权限
    4. 复制 token
    5. 推送时使用 token 作为密码
  
  方法2: 使用 SSH 密钥
    1. 生成密钥: ssh-keygen -t ed25519
    2. 添加到 GitHub: Settings → SSH keys
    3. 使用 SSH 地址: git@github.com:USER/REPO.git

【上传后】━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. 启用 GitHub Pages（可选）
     • Settings → Pages
     • Source: main 分支
     • 保存
     • 访问: https://YOUR_USERNAME.github.io/YOUR_REPO/
  
  2. 添加仓库描述
     • 点击仓库页面的 "Add description"
     • 填写项目简介
  
  3. 添加标签（Topics）
     • 点击 "Add topics"
     • 添加: education, science, interactive-learning 等
  
  4. 分享链接
     • 仓库地址: https://github.com/YOUR_USERNAME/YOUR_REPO
     • GitHub Pages: https://YOUR_USERNAME.github.io/YOUR_REPO/
