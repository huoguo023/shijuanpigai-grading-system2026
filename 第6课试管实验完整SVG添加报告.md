## ✅ 完成内容

### 1. 第4个知识点：试管空气柱实验（吹气）

#### 知识点内容
- **标题**：4. 试管空气柱实验 🧪
- **核心原理**：
  - 实验原理：对试管口吹气，让管内空气柱振动发声
  - 空气柱长度规律：
    - 空气柱长（水少）→ 振动慢 → 音调低
    - 空气柱短（水多）→ 振动快 → 音调高
  - 应用：管乐器（长笛、萧、单簧管）原理

#### SVG动画设计（360x180）

**4支试管展示**：
- 试管1：水最少（18px高）→ 空气柱最长 → Do（低音）
- 试管2：水较少（38px高）→ 空气柱较长 → Re
- 试管3：水较多（58px高）→ 空气柱较短 → Mi
- 试管4：水最多（78px高）→ 空气柱最短 → Sol（高音）

**渐变效果**（3个）：
- `glassGrad`：玻璃管透明质感（浅蓝渐变）
- `waterGrad`：水的立体效果（蓝色垂直渐变）
- `airGrad`：气流扩散效果（径向渐变）

**动态标注**：
- 橙色虚线标注空气柱长度
- 文字标注"长/较长/较短/短"
- 音符标注"Do/Re/Mi/Sol"
- 颜色编码：蓝色（低音）→ 紫色 → 粉色 → 红色（高音）

**交互动画**：
- ▶ 演奏试管琴：依次吹响4个试管
- ⏸ 停止：停止动画和音效
- 气流扩散动画（椭圆扩大，透明度渐变）
- 音符上升动画（♪上升并淡出）
- 音效播放（正弦波，Do=262Hz, Re=294Hz, Mi=330Hz, Sol=392Hz）

### 2. 第5个知识点：试管敲击实验（敲击）

#### 知识点内容
- **标题**：5. 敲击试管实验 🥢
- **核心原理**：
  - 振动对象：用木棍敲击试管壁时，振动发声的是**试管和水的整体**
  - 质量规律：
    - 水多 → 质量大 → 振动慢 → 音调低
    - 水少 → 质量小 → 振动快 → 音调高

#### SVG动画设计（360x200）

**4支试管展示**：
- 试管1：水少（28px高）→ 质量小 → 高音 ♪
- 试管2：水较少（48px高）→ 质量较小 → 较高 ♪
- 试管3：水较多（68px高）→ 质量较大 → 较低 ♪
- 试管4：水多（88px高）→ 质量大 → 低音 ♪

**渐变效果**（4个）：
- `glassGrad2`：玻璃管透明质感
- `waterGrad2`：水的立体效果
- `waveGrad`：振动波纹效果（黄色径向渐变）
- `stickGrad`：木棍质感（棕色渐变）

**动态元素**：
- 木棍敲击动画（向左移动并回弹）
- 振动波纹扩散（圆形从0扩大到25，透明度渐变）
- 音符上升动画（♪上升并淡出）
- 颜色编码：红色（高音）→ 粉色 → 紫色 → 蓝色（低音）

**交互动画**：
- 🥢 敲击演奏：依次敲击4个试管
- ⏸ 停止：停止动画和音效
- 音效播放（三角波模拟敲击声，880Hz→660Hz→494Hz→330Hz）

### 3. 对比总结表

#### 设计特点
- **表格式布局**（400x200）
- **蓝色渐变标题栏**
- **5列设计**：操作方式 | 振动对象 | 水量多 | 水量少

#### 内容对比

| 操作方式 | 振动对象 | 水量多（水位高） | 水量少（水位低） |
|---------|---------|----------------|----------------|
| 🌬️ 用嘴吹气 | 空气柱 | **高音** ♪<br>(空气柱短→快) | **低音** ♪<br>(空气柱长→慢) |
| 🥢 木棍敲击 | 试管与水 | **低音** ♪<br>(质量大→慢) | **高音** ♪<br>(质量小→快) |

#### 关键总结框
- 黄色背景，橙色虚线边框
- 强调三个要点：
  1. 吹气：振动的是**空气柱**，水多→空气柱短→高音
  2. 敲击：振动的是**试管+水整体**，水多→质量大→低音
  3. **同样的水量，操作方式不同，音高相反**！

## 🔑 核心知识点

### 为什么会"相反"？

**吹气实验**：
- 振动对象：空气柱
- 水多 → 空气柱短 → 振动快 → 高音
- 水少 → 空气柱长 → 振动慢 → 低音

**敲击实验**：
- 振动对象：试管+水整体
- 水多 → 质量大 → 振动慢 → 低音
- 水少 → 质量小 → 振动快 → 高音

**本质区别**：振动的对象不同！
