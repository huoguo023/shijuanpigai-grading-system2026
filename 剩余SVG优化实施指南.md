## 🎯 第一单元（声音）剩余优化清单

### 优先级1：核心教学动画

#### 1. 第1课《听听声音》
**待优化SVG：2个**

- **音符高低图**
  - 当前状态：简单折线图
  - 优化方向：
    - 添加渐变色彩（低音→高音：蓝→红）
    - 音符图标动画（上下浮动）
    - 五线谱背景
    - 标注清晰的音阶

- **音叉振动**
  - 当前状态：基础形状
  - 优化方向：
    - 金属质感渐变
    - 振动波纹（多层扩散）
    - 敲击动画
    - 声波可视化

#### 2. 第3课《声音是怎样传播的》
**待优化SVG：3个**

- **声波四面八方传播**
  - 优化方向：
    - 立体鼓（渐变鼓面+鼓身）
    - 多层同心圆波纹
    - 不同方向的听众
    - 波纹扩散动画

- **真空实验**
  - 优化方向：
    - 透明玻璃罩（渐变+高光）
    - 闹钟细节（表盘+指针）
    - 空气粒子逐渐消失动画
    - 声波减弱效果

- **纸杯电话**
  - 优化方向：
    - 3D纸杯（渐变+阴影）
    - 绷紧的线（振动动画）
    - 声波传导路径
    - 标注说明

#### 3. 第5课《声音的强与弱》
**待优化SVG：2个**

- **音量对比**
  - 优化方向：
    - 波形振幅对比（小波vs大波）
    - 颜色深浅表示强弱
    - 动画速度差异
    - 音量图标

- **鼓面振动实验**
  - 优化方向：
    - 立体鼓（已有参考：第2课）
    - 豆子跳动高度差异
    - 振幅可视化
    - 对比展示

#### 4. 第6课《声音的高与低》
**待优化SVG：2个**

- **音调对比**
  - 优化方向：
    - 波形频率对比（疏vs密）
    - 钢尺长短对比
    - 振动速度动画
    - 音高标注

- **频率演示**
  - 优化方向：
    - 波形图（正弦波）
    - 频率数值显示
    - 快慢对比动画
    - 颜色编码

## 🎨 统一优化标准（参考第三单元）

### 1. 渐变系统
```svg
<defs>
    <!-- 线性渐变 -->
    <linearGradient id="gradName" x1="0%" y1="0%" x2="0%" y2="100%">
        <stop offset="0%" style="stop-color:#color1"/>
        <stop offset="100%" style="stop-color:#color2"/>
    </linearGradient>
    
    <!-- 径向渐变 -->
    <radialGradient id="radialName">
        <stop offset="0%" style="stop-color:#color1"/>
        <stop offset="100%" style="stop-color:#color2"/>
    </radialGradient>
</defs>
```

### 2. 动画系统
```svg
<!-- CSS动画 -->
<style>
@keyframes animName {
    0% { transform: scale(1); }
    50% { transform: scale(1.1); }
    100% { transform: scale(1); }
}
</style>

<!-- SVG动画 -->
<animate attributeName="cx" 
         values="10;100;10" 
         dur="2s" 
         repeatCount="indefinite"/>
```

### 3. 交互系统
```javascript
function triggerAnimation() {
    const element = document.getElementById('elementId');
    element.setAttribute('attribute', 'newValue');
}
```
