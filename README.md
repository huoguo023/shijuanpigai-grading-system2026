# 小学科学四年级上册 - 互动学习网站

一个为小学四年级学生设计的科学学习网站，包含精美的SVG互动动画、即时反馈系统和闯关测试。

## 🌟 项目特色

### 📚 完整课程覆盖
- **第一单元：声音** - 8课完整内容
- **第二单元：呼吸与消化** - 8课完整内容  
- **第三单元：运动和力** - 8课完整内容

### 🎨 精美SVG动画
- 100+ 个交互式SVG动画
- 物理准确的科学演示
- 多层渐变和动态效果
- 支持点击交互和自动播放

### ✅ 即时反馈系统
- 答题即时音效反馈
- 彩色纸屑庆祝动画
- Toast提示消息
- 鼓励性文字提示

### 🏰 闯关塔测试系统
- 每课35-40道精心设计的题目
- 选择题、填空题、连线题、排序题
- 实时进度显示
- 错题解析功能

## 🚀 快速开始

### 在线访问
直接访问 GitHub Pages：
```
https://[你的用户名].github.io/[仓库名]/
```

### 本地运行
1. 克隆仓库
```bash
git clone https://github.com/[你的用户名]/[仓库名].git
cd [仓库名]
```

2. 打开主页
- 双击 `index.html` 或 `小学科学四上：科学大闯关总站.html`
- 或使用本地服务器：
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server
```

3. 在浏览器中访问
```
http://localhost:8000
```

## 📖 使用说明

### 主页导航
1. 打开 `小学科学四上：科学大闯关总站.html`
2. 左侧课题列表可快速跳转到各课
3. 点击课程卡片进入学习

### 课程学习
1. **知识复习**：查看SVG动画和知识点
2. **闯关塔**：完成35-40道测试题
3. **查看成绩**：获得分数和错题解析

### SVG动画交互
- 点击播放按钮启动动画
- 部分动画支持参数调节
- 观察物理过程和科学原理

## 🎯 学习目标

### 第一单元：声音
- 理解声音的产生、传播和特性
- 掌握音高、音量的影响因素
- 学习声音在生活中的应用

### 第二单元：呼吸与消化
- 认识呼吸系统和消化系统
- 了解营养物质和健康饮食
- 学习人体生理过程

### 第三单元：运动和力
- 理解力的概念和作用
- 掌握摩擦力、弹力的特性
- 学习简单机械原理

## 🛠️ 技术特点

- **纯HTML/CSS/JavaScript**：无需后端，易于部署
- **响应式设计**：自适应各种屏幕尺寸
- **SVG动画**：矢量图形，清晰流畅
- **Web Audio API**：原生音效，无需外部文件
- **本地存储**：可选的学习进度保存
